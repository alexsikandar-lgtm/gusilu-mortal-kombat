# GUSILU: MORTAL KOMBAT 🎮

¡Bienvenido a GUSILU: Mortal Kombat! Un juego de combate épico donde Gusilu, una criatura legendaria, debe enfrentarse a 7 poderosos adversarios para salvar el equilibrio dimensional.

## 🎭 CARACTERÍSTICAS

✅ **Sistema de Combate Estratégico**: 4 tipos de ataques diferentes
- Patada Rápida (Q) - Ataque ligero
- Puñetazo (W) - Ataque balanceado
- Patada Fuerte (E) - Ataque potente
- Movimiento Especial (R) - Ataque máximo

✅ **7 Personajes Únicos**
- GUSILU 🐛 - El protagonista
- KING KONG 🦍 - El coloso
- HUEVO SEQUAK 🥚 - El veloz
- PATATA NOBLE 🥔 - El titán terrenal
- PEDRO SÁNCHEZ 👨 - El político
- SANTIAGO ABASCAL 🧔 - El guerrero
- ABASCA OSCURO 👹 - El final boss

✅ **3 Niveles de Dificultad**
- FÁCIL (0.7x daño)
- NORMAL (1.0x daño)
- MORTAL (1.5x daño)

✅ **Cinemáticas de Encuentro**
- Introducción épica
- Diálogos dramáticos
- Secuencias de victoria

✅ **Sistema de HUD Profesional**
- Barras de vida dinámicas
- Log de combate en tiempo real
- Contador de rondas
- Estado de personajes

## 🎮 CÓMO JUGAR

1. **Abre el juego**: Carga `index.html` en tu navegador
2. **Selecciona dificultad**: Elige entre Fácil, Normal o Mortal
3. **¡A Combatir!**: Usa las teclas Q, W, E, R para atacar
4. **Derrota a todos**: Vence a los 7 adversarios para ganar

## 📋 CONTROLES

| Tecla | Acción |
|-------|--------|
| **Q** | Patada Rápida (80% daño) |
| **W** | Puñetazo (100% daño) |
| **E** | Patada Fuerte (130% daño) |
| **R** | Movimiento Especial (180% daño) |
| **ESPACIO** | Saltar cinemática |
